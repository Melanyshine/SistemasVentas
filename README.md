# Nombre del Proyecto:
Sistema de Ventas en C# con Arquitectura en Capas


## Descripción:
Este proyecto consiste en una aplicación de escritorio desarrollada en C# que permite gestionar un sistema de ventas completo. 

El sistema permite registrar y administrar categorías, productos, clientes y ventas, además de generar facturas y visualizar reportes. 

Está desarrollado aplicando el modelo de arquitectura en capas para una mejor organización y mantenimiento del código.


## Tecnologías Utilizadas:

- Lenguaje(s): C#
- Herramientas / Framework: .NET Framework, Windows Forms
- Base de datos: SQL Server
- Entorno o IDE: Visual Studio


## Uso o Ejecución:

1. Abrir el archivo SistemaVentas.sln en Visual Studio.
2. Verificar que la conexión a la base de datos esté correctamente configurada en el archivo App.config.
3. Ejecutar el proyecto presionando F5 o el botón "Iniciar".
4. Utilizar el menú principal para acceder a los módulos de categorías, productos, clientes y ventas.
